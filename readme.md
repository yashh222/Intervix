# 🤖 AI-Powered Conversational Interview Assistant

## 🚀 Overview

The InterVix is a smart, interactive web application designed to help users practice interviews in a realistic environment. It bridges the gap between *knowing answers* and *confidently expressing them under pressure*.

Instead of passive learning, this assistant actively engages you in conversation — asking questions, listening to your responses, and providing meaningful feedback to help you improve.

---

## 🎯 The Problem: The Interview “Action Gap”

Many people struggle during interviews not because they lack knowledge, but because they:

* Freeze under pressure
* Struggle to articulate answers clearly
* Lack real-time practice

> Knowing answers ≠ Communicating them effectively

Just like you can't learn cricket by only watching matches, you can’t master interviews without actually practicing speaking.

---

## ❌ Limitations of Current Practice Methods

* **Practicing with friends** → May not give accurate feedback
* **Watching videos** → Passive learning, no real interaction
* **Reading Q&A lists** → Memorization ≠ understanding
* **Paid mock interviews** → Expensive and limited access

---

## ✅ The Solution

This project provides a **fully interactive AI-driven interview experience** that:

* Asks relevant questions
* Listens to your responses
* Maintains conversation context
* Generates follow-up questions
* Provides structured feedback

## 🏗️ Project Flow

### 1️⃣ Start Interview

* Select a topic
* AI greets you
* First question is generated

### 2️⃣ Submit Answer

* Record your response
* Convert speech → text

* Generates a contextual follow-up

### 3️⃣ End Interview

* AI evaluates your overall performance

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Create virtual environment

```bash
python -m venv .venv
```

### 3. Activate environment

```bash
# Windows
.\.venv\Scripts\activate

# Mac/Linux
source .venv/bin/activate
```

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Add environment variables

Create a `.env` file:

```
GEMINI_API_KEY_KEY=your_api_key_here
MURF_AI_API_KEY=your_murf_ai_api_key_here
ASEMBLY AI_API_KEY=your_asembly_ai_api_key_here
```

### 6. Run the project

```bash
python main.py
```

---

## 🔐 Security Note

* Never commit your `.env` file
* Always use `.gitignore`
* Rotate API keys if exposed
